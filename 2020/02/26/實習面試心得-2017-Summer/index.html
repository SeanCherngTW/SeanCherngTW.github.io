實習面試心得 @ 2017 Summer

2017暑假是我大四畢業那年，再過2個月就要進入研究所，我選的實驗室算比較硬的，需要負責2學期的助教和不少產學合作案，因此我心知肚明碩一暑假不太可能去實習的，所以趁最後比較空閒的假期投了一些有興趣的公司和職缺。不巧的是，開始搜尋職缺後才發現很多公司都在JD表明要找「大三升大四」或「碩一升碩二」的學生，我的履歷很容易在第一關就被刷掉，因此就選擇性來說比前一年少了很多，幸好還是有公司願意找應屆畢業生擔任實習生，抱著姑且一試的心情，投了以下幾家公司的職缺（應該不只這四家，只是其他有投履歷的公司我已經想不起來了）：

  1. Acer - Marketing （一面後不錄用）
  2. Dell - Software Engineer（錄取）
  3. TrandMicro - Data Scientist（履歷無聲卡）
  4. Microsoft - RDAA（履歷無聲卡）

Acer - Marketing

在經歷了大三面試失敗的經驗、大四輔系資工系和實驗室接案後，發現自己的實力難以和本科資工系的學生相提並論，懷疑自己是否真的有能力得到一份軟體開發的工作機會。剛好資管系有學過管理與行銷相關課程，所以這次投遞履歷就不限制自己一定要找軟體職缺，因此投了Acer的行銷實習生職位。投遞履歷後不久就收到了面試通知，同時Acer Career粉絲專頁也公布了一份統計圖，大意是說Marketing+Business職缺總共收到了1,300份履歷，但被受邀面試的只有70人，我很幸運的是其中之一，也稍微增加了一點自信(?)。

Acer的面試形式很不一樣，以我面試的Marketing來說，所有面試者每5-6人被分成一組，在面試會場進行腦力激盪、團隊合作的創意挑戰(?，從團隊任務中判斷你適不適合這個職位。會場中有很多評分人員和主管，手上拿著筆和手寫板，記錄每一個應徵者的表現，說實話壓力還滿大的，尤其是看到他們動筆的時候你不知道他是不是在對你評分。印象中這次團體活動的關卡有：

  1. 棉花糖挑戰
  2. 每個人拿一條橡皮筋套在杯子上，另一端套在手上，6個人要從不同角度同時施力將杯子拿起來
  3. 用會場可見的任何東西組出最高的塔，但不能使用膠帶或任何能夠固定的工具將作品固定在地板、桌椅或牆壁上

我猜Acer大概是想從這個活動中看出每個人在團隊中的角色，例如你是主動跳出來當領導者或是默默當個跟隨者，你是主動調和意見衝突還是製造衝突，能不能想出有效率的解決方法等，模式有點像「誰是接班人」。我對於這類創意團康活動實在沒有太多idea，同組其他人討論熱烈時，我的參與度也比較低，多是在幫忙實作其他成員提出的方案。面試結束後，主管們帶所有應徵者到Acer的咖啡廳，除了可以享用Acer請客的咖啡之外，還可以跟其他應徵者或主管進行交流，算是一個打開人脈和了解公司的機會。從面試開始到結束總共花了3-4個小時，是我遇過最久的面試之一，疲累地回到宿舍後，自己也知道在活動參與的表現上沒有很好，心裡也大概有個底了。

結果：一面後不錄用

Dell Software Engineer

投遞履歷不久後，很快就收到面試邀請，一樣在面試一開始要使用投影片進行10分鐘英文自我介紹，碰巧的是，這次面試我的主管居然和一年前的主管相同，而主管也還記得我，算是一個緣份(?。在面談過程中得知這份職位的主要工作內容是將硬體測試的便利性進行提升，需要實作即時文本對照和即時通知的功能，主要工具使用Python+Java(Android)，剛好我的大學專題和大四選修有接觸到這一塊，所以過程中談得滿順利的，大多是在聊以往的專題經驗，還有熟悉的工具等，過程沒有太多技術問題。

面試完後過沒幾天，就收到了HR打來的錄取通知，也順利的進入Dell擔任兩個月的暑期實習生。

結果：Offer Get

結論

經過Acer的面試後，發現自己可能目前不太適合行銷或管理的職位，程式方面也比不上資工本科學生，雖然當時已經錄取同校的資工所，但也一度茫然不知道未來要往哪個方向走，或是我真的能在資工所順利畢業嗎？就在對未來感到困惑的時候，很感謝Dell能肯定我不起眼的經歷，讓我知道其實我的能力沒這麼差。
