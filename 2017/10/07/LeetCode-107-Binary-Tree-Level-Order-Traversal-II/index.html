LeetCode - 107. Binary Tree Level Order Traversal II

LeetCode Python

題目：
107. Binary Tree Level Order Traversal II
Given a binary tree, return the bottom-up level order traversal of its nodes' values. (ie, from left to right, level by level from leaf to root).

解釋：
輸入一棵樹，以Level order traversal走訪，並以buttom-up回傳每個Node的值

思路：
用類似BFS的概念，但必須記下每一個Node代表的Level。

  1. 首先只把root放進queue（Level 1只有root這個Node）
  2. 把queue裡面的節點全部pop掉，這些節點會代表同一個Level
  3. 針對剛剛被pop掉的每一個節點，把他們的左右子節點push到queue裡面
  4. 重複2, 3，直到所有節點都是葉子後停止
  5. 最後反轉ans list，就可以把原本的top-down改成buttom-up了
