LeetCode - 693. Binary Number with Alternating Bits

LeetCode Python

題目：
693. Binary Number with Alternating Bits
Given a positive integer, check whether it has alternating bits: namely, if two adjacent bits will always have different values.

解釋：
輸入一個整數，判斷這個整數轉為二進位後，每個bit的兩個相鄰bits是不是都和該bit不同
10101 -> True
11010 -> False
1 -> True

思路：
假設輸入的整數為n，n轉為二進位後，他的最大位元（最左邊的）一定為1
我們要找一個最大的i，使得2i < n
若n的輸出為True，根據i是奇數或偶數，以下兩條式子必有一個成立
n = 20 + 22 + 24 + ... + 2i
n = 21 + 23 + 25 + ... + 2i

  1. 首先找到i
  2. 符合條件的n可以改寫成
    n = 2i + 2i-2 + 2i-4 + ... + 21 (位元數目為奇數時)
    或
    n = 2i + 2i-2 + 2i-4 + ... + 20 (位元數目為偶數時)
  3. 因此可得
    n - 2i - 2i-2 - 2i-4 - ... - 21 = 0 (位元數目為奇數時)
    或
    n - 2i - 2i-2 - 2i-4 - ... - 20 = 0 (位元數目為偶數時)
  4. 若符合上述兩個等式之一，則回傳True，反之回傳False
