ML Lecture 10: Convolutional Neural Network

李宏毅機器學習筆記

圖像辨識使用CNN

  • 需要辨識的物體可能只佔了圖片的某一小塊（Convolution）
  • 同樣的pattern在圖片中的不同地方仍然代表一樣的意思（Convolution）
  • 圖片縮小後，仍然可以輕易地分辨出圖片中的物體（pooling）

CNN

  • Convolution & Filters

    • Filter代表你想要辨識的物體細節（E.g.紋路、筆跡、形狀等）
    • Filter會掃過整張圖片並對圖片的每一塊做內積，內積結果大的那一塊代表該處有我們想要辨識的物體
    • 做內積的過程就叫做Convolution
    • 多個Filters可以幫助我們辨識多個物體片段，進而拼湊出整個物體的形狀（例如：辨識數字5，某些Filters辨識橫槓、有些Filters辨識直槓、有些辨識圓弧等等，最後根據位置即可拼湊出這個數字是不是代表5）
    • Filters的參數數量為”Filters長 x Filters寬 x Channel數量”，和原始圖片大小無關，因為Filter在掃過整個圖片時，他的weight是共用的

  • MaxPooling

    • 根據圖片縮小後不變的特性，可以使用maxpooling來保留我們要辨識的目標物並丟棄其他不重要的pixel，讓機器在後續的處理上能更集中在目標物上

CNN Learning

  • 用gradient acsent找到一個x，使得他被Filter所啟動的範圍最大（內積的值越大）
  • 找到這樣的x目的是，讓下層的Convolution layer可以得到較接近目標物的圖片
