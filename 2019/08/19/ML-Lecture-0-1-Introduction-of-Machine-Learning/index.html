ML Lecture 0-1: Introduction of Machine Learning

AI v.s. ML v.s. DL

  • AI是達到人工智慧的「目標」
  • Machine Learning是達到AI的手段之一
  • Deep Learning則是Machine Learning的方法之一

AI的目的

  • 希望機器可以舉一反三，擁有自我學習能力，而不是完全靠人工手寫規則
  • 例如：給幾張貓狗的圖片並告訴機器哪些是貓哪些是狗，接著給其他沒看過的貓狗照片時，機器要有能力辨識

ML三步驟

  1. 準備一組function set
    • 可以把function set想成Model，不同的model代表不同function set
    • 同一個model代表同一個function set，參數中不同的weight和bias就代表同一個function set中不同的function
  2. 判斷function set中，function的好壞
    • 用validation data / testing data去看效能來判斷此function的好壞
    • 若效果很差，可能是
      • function set錯了（model不夠好）
      • function錯了（model是正確的，但超參數沒設定好導致train壞了）
  3. 找出最佳的function（這麼多種model，這麼多種參數，怎麼找？）
    • Gradient Descent（大略地從function set中找出一個行進方向）

訓練方法

  • Supervised Learning
  • Semi-supervised Learning
  • Unsupervised Learning
  • Transfer Learning
  • Reinforcement Learning

Task

  • Regression
  • Classification
  • Structured Learning
